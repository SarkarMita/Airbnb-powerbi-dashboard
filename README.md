## 📌 Project Overview
An interactive 3-page Power BI dashboard analysing Airbnb listing 
data across 10 global cities, covering pricing, host credibility, 
guest ratings, and review behaviour.

---

## 📊 Dashboard Pages

### 1. Overview Page
- KPI cards: Total Listings, Average Price, Superhost counts, Room types
- Cumulative Pareto chart ranking cities by listing volume
- Room type breakdown: Entire Place, Private Room, Hotel Room, Shared Room

### 2. Ratings Page
- 10 cities ranked by overall guest rating
- 🏆 Best rated: Mexico City (94.8) and Rio de Janeiro (94.6)
- ⚠️ Lowest rated: Hong Kong (89.7) and Istanbul (91.1)
- Cleanliness and value for money scored lowest globally
- Interactive drilldown toggle: Overall Rating ↔ Detailed Rating

### 3. Reviews Page
- 86% of reviewers wrote exactly one review
- 98.8% of all reviewers are one-time reviewers
- 🚨 Anomaly detected: One user wrote 283 reviews across 
  two Bangkok listings (Oct 2020) — flagged as potential bot
- Trust analysis: Host verification × Profile picture 
  across 4 credibility quadrants
